Exercise #2: Write, Compile, and Run TestGreeting

(Level 2)

Objective

In this exercise you will write, compile, and run the TestGreeting program from "A Basic Java Application" on page 1-8.

Directions

  1. Using a text editor, write the TestGreeting application program.

  2. Compile the program and correct any errors.

  3. Run the program using the interpreter.
