SL275: Module16: Networks

Exercise #2: Create a File Server

(Level 3)

Objective

In this exercise, you will have hands-on experience in the creation of a simple file server and client.

Directions

Work in pairs on this lab so you can use the host name of a machine other than your own. You are going to create a server and client pair and a program that requests a file from one of them.

  1. The server template is in the FileServer.java file. Develop the methods that will enable the server to receive a string file name from the client and attempt to open it, and pass it back to the client on the socket.

  2. The client template is in the ReadFile.java file. The client program takes a string file name argument and sends it to the server, and then waits for the server to send either an error response or the file itself.
