SL275: Module14: Threads

Exercise #1: Use Basic Threads

(Level 1)

Objective

In this exercise you will have hands-on experience in the creation and running of threads.

Directions

The purpose of this lab is to create three threads and run them at the same time. While are running they will be printing out their name into a textarea. by observing what is printed into the text area you can observe how the threads run and in which order.

  1. Create a class called PrintMe which implements Runnable.
    In the run method, loop 10 times: print the name of the current thread and then sleep for 2 seconds.

  2. Create a program called TestThreeThreads.
    1. In the main, create an instance of PrintMe.
    2. Create three threads using the PrintMe object.
    3. Name each thread using the setName method.
    4. Start each thread.

  3. Compile and run the TestThreeThreads program.

Question: can you explain the behavior of your program?
