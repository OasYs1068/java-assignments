SL275: Module15: Advanced I/O Streams

Exercise #3: Create a Simple Database Program

(Level 3)

Objective

In this exercise you will use random access files to implement a simple database program.

Directions

  1. Create an application called DBTest.java that emulates a small database program that stores and retrieves records relating to products. Use the RandomAccessFile class and a flat file.

  2. Records in the database should consist of string names and integer quantities.

  3. Your program should allow the user to display a record, update a record, and add a new record.
