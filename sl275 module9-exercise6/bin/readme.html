SL275: Module9: Text-Based Applications

Exercise #6: Build an Archive of the Bank Project

(Level 2)

Objective

In this exercise you will create a JAR file of the Banking project. You will then run the reports program using only the JAR file.

Directions

Start by changing your working directory to SL275/mod09/exercise6 on your computer. Copy the previous Banking project files.

  1. Compile and run the TestBanking program.

  2. Build a Java archive file using the following command: 
    jar cvf banking.jar banking/domain/*.class banking/reports/*.class

  3. Remove all of the class file in the Banking Project packages. 
    rm banking/domain/*.class banking/reports/*.class

  4. Run the TestBanking program using the JAR file: 
    java -classpath banking.jar:. TestBanking

  5. Why was the :. important in the specification of the classpath option?
